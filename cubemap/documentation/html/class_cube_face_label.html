Detailed Description

A holder (a QLabel) for a face of a cube map. This UI element accepts a drag'n'drop of image files and shows a scaled down version of the image. Images can be swaped between two CubeFaceLabel. Information about the source file dimensions and the number of frames in the sequence are displayed on top of the image.
