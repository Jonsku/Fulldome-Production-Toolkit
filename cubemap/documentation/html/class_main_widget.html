Detailed Description

The main widget initializes all the modules of the application and handles project saving and loading from the File menu.

void MainWidget::load ( ) [slot]

Opens a file dialog for the user to select a JSON file and load the project. At the moment, all unsaved changes will be lost.

void MainWidget::save ( ) [slot]

Save the current project to a JSON file. The previous saved file will be overwritten. However, if it is the first time the project is saved, saveAs() will be called instead.
