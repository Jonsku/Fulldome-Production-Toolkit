\subsection{\-Detailed \-Description}
\-A collection of inline functions to find frames of a same sequence based on files name. 

\-Cube\-Animator accepts any file name in the format \char`\"{}name.\-extension\char`\"{}. \-Furthermore, it will look for sequence of files named \char`\"{}whatever\-Xwhatever.\-extension\char`\"{} where \-X is a numeral. \-So if you drop a file called \char`\"{}name.\-png\char`\"{} on one of the face, then files like \char`\"{}name0.\-png\char`\"{} or \char`\"{}name21.\-png\char`\"{} will be used in the sequence. \-Also if you load a file called \char`\"{}name1.\-png\char`\"{}, it will detect all files named \char`\"{}name\-X.\-png\char`\"{}. \-If it is called \char`\"{}name2top.\-png\char`\"{}, then it will detect all files named \char`\"{}name\-Xtop.\-png\char`\"{}. \-In case you a have file whose name contains two or more numerals, such as \char`\"{}name1\-\_\-1.\-png\char`\"{}, \-Cube\-Animator will detect files names \char`\"{}name1\-\_\-\-X.\-png\char`\"{} as being part of a the same sequence.
